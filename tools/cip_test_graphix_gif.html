Local CIP Diagnostic (avg mask + transitions → one GIF)

Loads img/test_graphix/b1.png, b2.png, b3.png from this repo, builds the shared mask as the pixel-wise average (plus noise + block shuffle), generates mask↔image dissolve transitions, then stitches a single diagnostic GIF: mask → m2i(b1) → b1 → i2m(b1) → mask → ... → b3 → i2m(b3) → mask.

Run via a local server (recommended): python -m http.server from the json-builder-app folder, then open http://localhost:8000/tools/cip_test_graphix_gif.html.
